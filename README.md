# 电磁专家成长手册

The path to an EM expert

## 愿景

我们致力于打造一支有温度、有热情、有特色的电磁研究团队，培养优秀人才。

## 冷水

“专家之路”很难（不限于“电磁专家”）。为了实现这一目标，需要你：

保持坚定的必胜信念，

培养发自心底的热忱，

投入长时间持续专注的精力，

开展深入的理论思考和丰富的一线实操，

经历不计其数的挫折甚至失败，

反复锤炼学习/研究方法和技巧，

忍受导师和朋辈们随时随地的“瞎指挥”，

......

## 态度

道德操守方面

- 诚信：在工作和生活中坚守诚信原则，不可违背。

团队合作方面

- 奉献精神：秉持 “人人为我，我为人人” 理念，以成年人心态积极投入团队工作。
- 责任心：认真完成自己应允的工作，特别是协作性工作，保质保量完成。
- 合作态度：平和友善待人，积极培养合作精神，遇到矛盾理性交流。

自我成长方面

- 认真努力：坚信一分耕耘一分收获，视勤奋为安身立命的根本。
- 好奇心：保持好奇心，头脑开放，积极学习新理论和新技术。
- 独立思考：以事实或常识为依据形成合理观点，避免人云亦云，且认识到观点可随时间改变，不必执着于旧观点。杜绝说“别人这样，所以我也这样”。
- 自我认知：勇于承认自身不足，明白没有蠢问题。

## 习惯

自我提升习惯

- 简历意识：经常提醒自己，如果写求职简历可以呈现哪些亮点。
- 保持输出：认识到没有输出的输入可能是假象，持续进行输出。

沟通协作习惯

- 积极沟通：积极主动进行沟通，非节假日不应无故长时间不回应，避免不礼貌和不专业的行为。
- 坦诚交流：在学术讨论中尽量坦诚，发现问题要指出（包括老师的错误）。
- 及时求助：遇到自己无法解决的问题，尽早让老师或同伴介入，不独自闷头做或回避。

任务管理习惯

- 任务闭环：养成任务闭环习惯，有方案直接提供，暂时没有则给出下一步思考。
- 工作记录：在假期以外坚持写研究周报，记录研究过程和思考。

## **任务**

所谓“过一遍”，是希望大家从头到尾翻一遍，能看懂多少就是多少，不追求一遍看透，但希望可以在头脑中留下框架性的印记。专业书籍，难度普遍较高，需要翻阅多次。每次遇到专业问题，都需要翻书再认真详读。

### **本科和硕士一年级**

1. 完成《Learn Python the Hard Way》的前45章的学习（在线书 https://github.com/ubarredo/LearnPythonTheHardWay ）；
2. Stanford的Writting in Science过一遍（https://www.bilibili.com/video/BV1L7411n73j ）；
3. Kraus的天线书过一遍（配套练习 https://tbc-python.fossee.in/book-details/251 ）；
4. Pozar的微波工程过一遍（配套练习 https://tbc-python.fossee.in/book-details/814 ）；
5. 阅读近30年来IEEE APS的获奖文章（ https://ieeeaps.org/about-the-transactions/best-paper-awards ），并汇总成表格，配上文字介绍。

### **硕士二、三年级**

1. MIT Walter Lewin的电磁学视频过一遍（ https://www.bilibili.com/video/BV124411R7r2 ）；
2. 费曼物理学讲义的电磁部分过一遍（ https://www.feynmanlectures.caltech.edu ）；
3. Balanis的天线书过一遍（配套练习 https://tbc-python.fossee.in/book-details/876 ）；
4. 梁昌洪的简明微波过一遍（配合他讲的微波技术视频： https://www.bilibili.com/video/av90034984 ）；
5. 看懂Chu极限推导《PHYSICAL LIMITATIONS OF OMNIDIRECTIONAL ANTENNAS》；
6. 搞懂CMA方法，学会使用仿真软件的CMA功能作特征模式分析；
7. 选一种算法，实现dipole/patch/slot天线的电磁计算；
8. 总结近三十年来天线领域的大牛名单，阅读他们在IEEE Proceeding的天线文章，和TAP的受邀请天线文章（数量不多，几乎全是精品）。

### **博士**

1. 顾樵的数学物理方程过一遍，或者姜颖的数学物理方程视频过一遍（https://space.bilibili.com/548272388 ）；
2. Gilbert Strang的Introduction to Linear Algebra视频过一遍（ https://www.bilibili.com/video/BV1zT4y1u7c7 ）；
3. Harrington的时谐电磁场过一遍；
4. Kildal的Fundamentals of Antenna Engineering过一遍；
5. 陈志豪的应用电磁学讲义过一遍；
6. 阅读天线领域重要研究组的有影响力的过往文章，形成一份“门派”地图；
7. 每一期新出的TAP和AWPL都快速刷一遍，并争取形成简要文档记录（比如新天线的结构图汇总）；
8. 经常问自己：提交教职申请的时候，打算怎么用一句话总结自己的研究特色（研究方向）？

## **组会**

1. 每周两次组会，一次文献分享，一次研究进度汇报。每人每周做一次报告，文献和进度报告交替进行；
2. 如无特殊情况，按时参加组会（现场和线上），无故不参加或者迟到算缺勤；
3. 大四毕设阶段本科生和硕士研究生必须讲，哪怕只有一页ppt也要报告，无特殊情况不可跳过；
4. 报告要有逻辑，不能意识流输出。建议的报告内容：当前在研究什么主题，报告在尝试解决什么具体问题，自己的分析思路和解决方案，验证方法和结果，当前结论，下一步的计划；
5. 硕士的组会报告用英文写，可以汉语讲；
6. 限制时间，每人在5分钟之内讲完。内容实在多的话，等其他人全讲完再展开；
7. 常规进度组会分两个环节：第一部分，硕士研究生介绍当周的工作和进展、以及接下来的预期工作。讲完的可以先走，不过鼓励听完；第二部分，本科生（毕设学生为主）介绍学习/研究进展、问题、复现工作等，讨论可能的解决办法；
8. 常规组会的两个环节之外，可以事先安排一些偶发性长时段内容。比如学术会议彩排，答辩彩排等；
9. 考虑到经常有新人加入，不定期做自我介绍：家乡在哪里？有几个兄弟姐妹？经历过什么挑战？个人爱好？讲一个笑话（别人不笑很正常）。
## 课题

1. 组内研究主题相对集中，主要在天线/电磁方向，也会在这个方向上持续投入，打造出团队的研究特色；
2. 在研究主题方面，建议听取导师的意见。导师们熬过了正规学术训练的完整周期，并通过了进入中大的窄门，才有机会指导学生。比起二十多岁的学生，导师们在知识积累，经验方法和视野等方面，具有相对优势，可以提供更好的指导。如果不是这样，全世界通行的研究生导师制便没有存在价值。另外，如果你相信导师不傻不坏，也可以相信他希望你拥有一个美好的未来，并会努力助你实现，因为学生是导师最重要的“作品”；
3. 项目主要分三类，自由探索，纵向课题，横向课题；一般来说，从左到右，研究的约束条件、实用价值和工作节奏，逐渐增强；
4. 老师们会谨慎选择项目，只做对团队长期发展有利的工作。不论什么类型的项目，都是为了研究组的愿景服务，确保大家可以通过项目研究，探索科学和工程中的关键问题，形成有价值的成果，体现为发表高质量文章；
5. 纵向课题和横向课题是研究内容的主要来源，资源会尽量集中到优势方向和战略方向。自由探索内容比例相对低很多，不会成为主流，只适合少数学术潜力股。我们希望打造成有特色的团队，而不是各自为战的团伙；
6. 无论纵横，一旦选择，请对自己的角色负责。对学生来说，项目既是研究的机会和支撑，也是大家走向社会前的一次体验和“排练”；对老师来说，项目既是对科学/工程前沿的深度探索，也是以名声作保的郑重承诺。出色完成项目，是争取下一个优质机会（比如dream work工作或者dream school博士机会）的最好资本，有面试经验的同学对此感触最深。

## 规矩

1. 多数时候，导师都是对的，即使不对，也不见得离谱太多。如果不认同导师的意见，可以尝试收集证据说服导师（原则上最多一次），但不能阳奉阴违，也不建议直接对着干。如果自己天赋异禀且勤奋异常，欢迎带飞导师。否则，不建议自己定义怎么指导自己；
2. 来去自由，欢迎本科生进组体验。对进组体验的同学，不说不做的静默体验期一般为1个月（最多不超过2个月），之后要上手做研究（新加入的研究生没有静默期）；
3. 工作日，请研究生同学在10:30am之前（当前要求）到办公室开展学习和研究。恶劣天气、生病、有课、临时有事或者导师认可的其他特殊情况例外。大段时间不在办公室出现，显然不是我们团队成员应该有的状态；
4. 假期和周末自由安排，不作要求。工作日，默认不离开学校，不无故“失联”，离校需请假。需要工作日离校的同学，请按照学籍管理规定要求，事先提出请假申请；
5. 人人为我，我为人人。为了团队的健康运作和发展，需要每位同学参与到团队的日常/突发工作中，一起建设，没有例外；
6. 学习/研究讨论的群组实施动态管理，仅面向现有成员开放，离开的成员自动被转移到“组友群”；
7. 严守学术规范，绝不可以触犯任何学术道德底线；
8. “礼物”只能从老师流向学生，不能反向。

## 招生

1. 博士：目前，本组博士招生名额有两处来源，中山大学和香港城大。中山大学的招生名额很少，并非每年都有；城大的招生名额几乎每年都有，但数量也很少。无论哪个来源，申请人都需要完成正规的多轮面试流程，并面对多位候选人的竞争。如果想要攻读博士学位，请提前联系本团队老师；
2. 硕士：常年面向本科生提供科研体验机会（大创，工程训练等，或者体验组会），并欢迎大家留在团队读研。统计来看，团队的硕士研究生主要来自团队的本科生群体，团队内绝大多数本科生，都已成功保研/考研，并将继续在团队内开展研究工作；
3. 团队目前共有四位导师，按加入中大的时间顺序，先后为梁国华（博导、硕导），杨楠（博导、硕导），陆凯（硕导）和胡鹏飞（硕导）。四位导师深度合作，共同指导组内学生。
